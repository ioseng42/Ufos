ÂµFramework for @Observed Signal.

## What?

Building data-driven UI without `Rx`.

*All you need for a reactive UI is an **@Observed property** and an **event broadcaster**.*

That's all, folks!

### Features

* Access control
* Automatic memory management
* `DispatchQueue` support
* Filtering
* `Keypath` support
* Strongly typed
* Thread safe
